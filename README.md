# APIs of the todo list with token user autantication.


## Getting Started

First, add the node modules:

```bash
npm i
```

Run the development server:

```bash
npm run start:dev
```

## APIs

- Create user API.
- Login user API.
- Get user by Id.
- List all user.
- Forgot Password.
- Edit User.
- Role based Auth.
- Create To Do.
- List of all todos.
- Get todo By user Id.
- Edit Todo.


## Technologies Used

- Nest.js 
- Postgres SQL

## .env file

```bash
DB_CONNECTION=postgres
DB_DATABASE=<database name>
DB_HOST=localhost
DB_PORT=5432
DB_USERNAME=<db username>
DB_PASSWORD=<db password>
DB_SYNC=false
DB_LOGGING=true
JWT_KEY=key
JWT_EXPIRE=3600

```
